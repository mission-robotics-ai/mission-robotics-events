# mission-robotics-events

Shareable assets & ops for **Mission Robotics** community events — open houses, mixers, networking nights.

Hackathon-specific assets (judge/sponsor/hacker cards) live in [`embodied-metal-ops`](https://github.com/mission-robotics-ai/embodied-metal-ops).

This repo is public so assets are linkable — drop a raw URL into a Luma cover or hand it to a co-host.

## Structure

`assets/<event>/` — rendered cards per event, at OG (1200×630) and square (1080×1080).

### `assets/open-house/`
**Physical AI Open House** — Thursday, July 3 · Mission Robotics × The Residency.
- `event-openhouse-og.png` · `event-openhouse-square.png`

## Grabbing a raw URL

```
https://raw.githubusercontent.com/mission-robotics-ai/mission-robotics-events/main/assets/open-house/<file>.png
```
